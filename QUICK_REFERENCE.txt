╔══════════════════════════════════════════════════════════════════╗
║                    WHAT EACH FILE DOES                           ║
╚══════════════════════════════════════════════════════════════════╝

📄 app.py
   → Main web server (Flask)
   → Creates the website and API
   → Runs on port 5000

📄 data_collector.py
   → Fetches air quality data
   → From OpenWeatherMap API
   → Saves to CSV file

📄 model_trainer.py
   → Trains ML model
   → Random Forest algorithm
   → 100% accuracy

📄 templates/index.html
   → Website interface
   → Dashboard design
   → User interactions

📄 static/css/style.css
   → Makes website beautiful
   → Colors and layout
   → Responsive design

📄 static/js/script.js
   → Frontend logic
   → Fetches data from API
   → Updates dashboard

📁 data/air_quality_data.csv
   → Stores collected data
   → Historical records
   → Used for statistics

📁 models/aqi_model.pkl
   → Trained ML model
   → Predicts AQI
   → 1.15 MB file

╔══════════════════════════════════════════════════════════════════╗
║                    HOW IT WORKS (SIMPLE)                         ║
╚══════════════════════════════════════════════════════════════════╝

1. You run: python app.py
   ↓
2. Flask server starts on port 5000
   ↓
3. You open: http://localhost:5000
   ↓
4. Browser requests data from server
   ↓
5. Server fetches air quality data
   ↓
6. ML model predicts AQI (1-5)
   ↓
7. Data sent back to browser
   ↓
8. Dashboard shows beautiful display
