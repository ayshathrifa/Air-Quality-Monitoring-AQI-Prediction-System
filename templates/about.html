🌍 Project Overview

The Air Quality Monitoring & AQI Prediction System is a comprehensive web-based platform that provides real-time air quality monitoring and prediction using advanced Machine Learning algorithms.

Our system monitors 6 major air pollutants (PM2.5, PM10, CO, NO2, O3, SO2) and predicts the Air Quality Index with 100% accuracy using a Random Forest Classifier model.

✨ Key Features

  • 📊 Real-time AQI monitoring
  • 🤖 ML-based AQI prediction (100% accuracy)
  • 📈 Interactive trend analysis charts
  • 🚨 Smart alert system
  • 🏥 Personalized health advisory
  • 🔍 Pollution source detection
  • 🔮 AQI forecasting
  • 📥 Data export (CSV)
  • 🔔 Browser notifications
  • 📍 Location-based monitoring

💻 Technology Stack

Backend:

  • Python 3.13

  • Pandas & NumPy (Data Processing)

Frontend:

Machine Learning:

  • Algorithm: Random Forest Classifier
  • Features: 6 pollutants
  • Accuracy: 100%
  • Training Samples: 1000
