✨ Key Features

📡

Real-Time Monitoring

Live air quality data for any location in India using OpenWeatherMap API with intelligent fallback

🤖

Multi-Model ML

Compares Random Forest, XGBoost, Gradient Boosting and Logistic Regression — best model auto-selected

⏱️

24-Hour Forecast

Time-series AQI projection for the next 24 hours using rolling mean with diurnal traffic patterns

📅

Seasonal Trends

Monthly PM2.5 and AQI analysis across 365 days revealing winter peaks and monsoon improvements

🗺️

Pollution Heatmap

Day-of-week × hour-of-day PM2.5 heatmap to identify high-pollution windows at a glance

🏥

Health Advisory

Personalized recommendations — outdoor activity guidance, mask advice, and sensitive group alerts

🔍

Source Detection

Identifies likely pollution sources — traffic, industrial, construction dust, or biomass burning

🗄️

SQLite Database

All readings stored in a structured SQLite database with full CSV export support

⚙️ How It Works

1
📥

Data Collection

Fetches PM2.5, PM10, CO, NO₂, O₃ and SO₂ from OpenWeatherMap API or uses location-based demo data

2
🗄️

Storage

Every reading is saved to SQLite with timestamp and coordinates, seeded with 365 days of historical data

3
⚙️

Preprocessing

Features are scaled using StandardScaler; AQI labels are derived from EPA PM2.5 breakpoints

4
🤖

Model Training

Four ML models are trained and compared; the highest-accuracy model is saved and used for predictions

5
📊

Prediction & Forecast

Best model predicts current AQI; rolling-mean time-series projects the next 24 hours with diurnal patterns

6
📈

Visualization

Results displayed as interactive charts — model comparison, seasonal trends, heatmap, and live trend graphs
