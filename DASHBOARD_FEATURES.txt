🎯 API ENDPOINTS
═══════════════════════════════════════════════════════════════════

GET  /api/current      → Current air quality data
GET  /api/history      → Historical data (last 50)
GET  /api/trend        → Trend data for chart (last 20)
GET  /api/forecast     → AQI forecast (next hours)
GET  /api/stats        → Statistics
GET  /api/download     → Download CSV file
POST /api/predict      → Custom AQI prediction
