🎯 HOW TO TEST UPDATE BUTTON
═══════════════════════════════════════════════════════════════════

STEP 1: Open Browser Console
   Press F12 → Console tab

STEP 2: Enter Coordinates
   Example: Delhi
   Latitude: 28.6139
   Longitude: 77.2090

STEP 3: Click Update Button

STEP 4: Check Console
   You should see:
   "Location updated to: 28.6139, 77.2090"
   "Fetching data for: 28.6139, 77.2090"
   "Received data: {pm2_5: 125.8, ...}"

STEP 5: Verify Changes
   - AQI changes from 3 to 4
   - Color changes from Orange to Red
   - PM2.5 changes from 45.5 to 125.8
   - Health advice updates
   - Pollution source updates
   - Notification appears (for Delhi)

═══════════════════════════════════════════════════════════════════

🐛 IF UPDATE BUTTON NOT WORKING
═══════════════════════════════════════════════════════════════════

CHECK 1: Browser Console (F12)
   Look for errors in red

CHECK 2: Network Tab (F12 → Network)
   Click Update → See if API call is made
   Should see: /api/current?lat=28.6139&lon=77.2090

CHECK 3: Hard Refresh
   Press: Ctrl + Shift + R
   Clears cache and reloads JavaScript

CHECK 4: Verify Input Values
   Make sure numbers are entered correctly
   No extra spaces or characters
