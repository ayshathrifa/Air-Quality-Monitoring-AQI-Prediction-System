🎯 FEATURE 3: AQI NOTIFICATION SYSTEM
═══════════════════════════════════════════════════════════════════

WHAT IT DOES:
- Sends browser notifications when AQI becomes dangerous
- Alerts when AQI reaches 4 (Unhealthy) or 5 (Very Unhealthy)
- Works even when browser is in background

NOTIFICATION EXAMPLE:
┌─────────────────────────────────┐
│ ⚠️ AQI Alert                    │
│                                 │
│ Air Quality is Unhealthy!       │
│ AQI: 4                          │
│ Avoid outdoor activities.       │
└─────────────────────────────────┘

HOW IT WORKS:
1. Browser asks for notification permission on first load
2. System monitors AQI continuously
3. When AQI ≥ 4, notification is sent
4. User gets alert even if not looking at page

REQUIREMENTS:
- User must allow notifications (browser will ask)
- Works on Chrome, Firefox, Edge
- Desktop and mobile supported

🎯 FEATURE 4: POLLUTION SOURCE DETECTION
═══════════════════════════════════════════════════════════════════

WHAT IT DOES:
- Analyzes pollutant ratios to identify pollution sources
- Uses AI-based logic to detect source type
- Shows confidence percentage

DETECTION LOGIC:

1. TRAFFIC EMISSION
   - High NO2 (>40) + High CO (>600)
   - Confidence: 85%
   - Common in urban areas

2. INDUSTRIAL POLLUTION
   - High SO2 (>20) + High PM10 (>100)
   - Confidence: 78%
   - Near factories/industries

3. CONSTRUCTION DUST
   - PM10 > 2× PM2.5 + Low NO2 (<30)
   - Confidence: 72%
   - Building sites, road work

4. BIOMASS BURNING
   - High PM2.5 (>60) + High CO (>700)
   - Confidence: 80%
   - Crop burning, wood fires

5. MIXED SOURCES
   - When no clear pattern
   - Confidence: 65%
   - Multiple pollution sources

EXAMPLE OUTPUT:
┌─────────────────────────────────┐
│   Traffic Emission              │
│   Confidence: 85%               │
└─────────────────────────────────┘

WHERE TO SEE:
- New "🔍 Pollution Source Analysis" card
- Updates with each data refresh
